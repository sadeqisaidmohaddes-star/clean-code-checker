Zero-dependency Python

Find the code that deserves attention.

Clean Code Checker reads a GitHub repository, scores its maintainability, and points to specific files and lines worth improving.

View source on GitHub Setup guide
Why this page is not the analyzer: GitHub Pages serves static files and cannot run the Python analysis engine. Clone the repository and launch it locally to perform real scans; repository code and optional tokens stay on your machine.

Concrete findings

Reports filenames, line numbers, severity, and practical explanations instead of a mystery score.

Broad checks

Covers complexity, function and file size, formatting, comments, duplication, and maintainability signals.

No package install

The application uses Python's standard library and talks directly to the GitHub REST API.

Run it locally

Python 3 is the only runtime requirement. The app opens a local web interface at port 8000.

git clone https://github.com/sadeqisaidmohaddes-star/clean-code-checker.git
cd clean-code-checker
python app.py
